Drumbeat Code Test

This is a codetest intended to test core front end development

The goal is to take the figma design in the root of this project `drumbeat-codetest.fig` and to recreate it in a responsive example

Figma can used entirely in browser for free at www.figma.com, we have provided and export file rather than a shared link for anonymity

The API with corresponsing data is available here https://codetest.drumbeat-server.co.uk/wp-json/wp/v2/posts?_embed&per_page=100 this is open CORS so is greenlit for ajax / fetch / axios to use. Please use this api, any missing data such as cetegory etc from the API feel free to use palceholder text

The reduced build script to setup a local development environment is in included and setup instructions are located in the readme.md

Please take no longer than an hour to complete this after setting up the node / gulp build tools, we're looking for code quality not code completion, we would much rather see well executed incomplete code alongside a note of next steps at the end of the hour

We are happy to recieve a fork of this repository or a zip file of the completed code test.
